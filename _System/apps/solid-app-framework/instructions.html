Description

This app is meant to make improving your meta tags across your website quicker and easier. This app will scan pages and will show you the relevant data associated with the page so you can discover which items need to be updated.

Details

This app uses the following default guidelines for evaluating a page:

  1. Page Title exists and is between 8 and 70 characters
  2. Meta Description exists and is less than 160 characters
  3. URL exists and is less than 70 characters
  4. URL should not contain a file extension, expect for index files (ex .html)

What gets evaluated?

Pages that are live and not set to be excluded from search engines

Important

This app scans the pages using the API and does not scan the final output of the page. This means that this app does not check that the Page Title and Meta Descriptions are actually being outputted on the pages correctly.

Settings

>Using the settings tab, you can change the values used to evaluate the page data: page title length, meta description length, and URL length

Limitations

This app only scans up to 500 pages. This means any pages after 500 will get cut off and will not display in the app.
